HTML & CSS Cheatsheet

A concise guide on HTML elements, attributes, and CSS properties.

HTML Elements & Attributes

  • <headings> - Define titles and subtitles (<h1> to <h6>)
  • <paragraphs> - Blocks of text (<p>)
  • <head> / <body> - Document structure
  • <meta> - Metadata like charset and viewport
  • <links> - Hyperlinks (<a>)
  • <tables> - Tabular data (<table>, <tr>, <td>)
  • <list> - Ordered and unordered lists (<ul>, <ol>, <li>)
  • <forms> - User input forms (<form>, <input>, <button>)
  • Other Attributes
    • page title - Set inside <title>
    • favicon - Small icon in the browser tab
    • text formatting tags - <b>, <i>, <strong>, <em>
    • images - <img> tag for pictures

CSS Cheat Sheet

  • Selectors - Target HTML elements (element, .class, #id)
  • Comments - /* comment */
  • Colors - Named colors, HEX, RGB, HSL
  • Measurements - px, em, rem, %, vh, vw
  • Properties & Values
    • background - background-color, background-image
    • border - width, style, color
    • padding & margin - spacing inside and outside elements
    • opacity - transparency level
  • Layout
    • height / width
    • text & fonts - font-family, size, weight
    • icons
    • list & tables - styling lists and tables
    • position, z-index, overflow - control stacking and visibility
    • float, inline, block
  • Navigation Bar - Styling links
  • Dropdowns - Menus that expand/collapse
  • Image Gallery - Layout and responsive images
